# @gabriel9x9

前端公共的函数、组件、hooks
> 单仓库多项目

## use

```
cd packages/
pnpm run build
pnpm publish --no-git-checks

pnpm add @gabriel9x9/shared -w
```

## 软件架构

> [pnpm](https://pnpm.io/zh/)、 [react18](https://react.nodejs.cn/)、[typescript](https://www.typescriptlang.org/zh/)
